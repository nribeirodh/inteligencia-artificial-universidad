# Inteligencia Artificial – Prácticas UOC

Este repositorio no es un súper proyecto de IA, es simplemente donde estoy guardando
las prácticas de la asignatura de **Inteligencia Artificial** del grado de Ingeniería
Informática (UOC), pero ordenadas y explicadas.

La idea es que se vea cómo he trabajado en la asignatura: cómo represento
conocimiento, cómo razono con reglas, cómo uso lógica difusa y cómo aplico modelos
clásicos de aprendizaje supervisado.


# PEC2. Representación del conocimiento

Aquí trabajo con:

- Un **sistema de marcos** para clasificar plantas (clases, subclases, campos, herencia…).
- Un **sistema basado en reglas** para hacer previsiones meteorológicas y otro ejemplo propio.
- Encadenamiento hacia delante y hacia atrás y resolución de conflictos entre reglas.

Es la parte lógica/simbólica de la IA.

# PEC3. Lógica difusa

En esta pec diseño un **sistema difuso tipo Mamdani**:

- Defino las variables de entrada y de salida con sus etiquetas lingüísticas
  (bajo, medio, alto, etc.).
- Construyo las funciones de pertenencia y el conjunto de reglas.
- Hago todo el proceso de inferencia y defuzzificación para un ejemplo concreto.

Aquí el foco está en el razonamiento aproximado: no todo es “verdadero/falso”,
sino grados de pertenencia.

# PEC4. Árboles de decisión y redes neuronales

Esta PEC está centrada en **aprendizaje supervisado**:

- Entrenamiento de un **árbol de decisión** y de una **red neuronal** sobre un mismo problema.
- Comparación de resultados (matrices de confusión, métricas, etc.)
- Comentarios sobre cuándo tiene sentido usar cada modelo.
